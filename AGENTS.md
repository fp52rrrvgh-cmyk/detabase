# AGENTS.md

This repository is maintained for a personal operations system, with finance as the first module.

## Working Rules

- Confirm the current working directory before editing.
- Keep changes inside the requested scope.
- Prefer `rg` for search.
- Do not add app code, database schema, Supabase configuration, dashboard code, or legacy Sheets/GAS logic unless explicitly requested.
- Do not introduce version labels or release decisions.
- Do not describe the project as production-ready.

## Source of Truth

- `docs/PROJECT_STATE.md` records the current known state.
- `docs/NEXT_ACTIONS.md` records the next planned work.
- `docs/DECISIONS.md` records accepted decisions.
- `docs/DO_NOT_DO.md` records prohibited work.
- `docs/RUNBOOK.md` records operating steps once they exist.
- `docs/FINANCE_SPEC.md` records finance-domain requirements once they are defined.
