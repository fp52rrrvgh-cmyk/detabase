# AI 建議閉環 v1 — 施工圖紙

**狀態**: 小馬提案 → Codex 審查/修正 → ✅ 共識 → ⏸️ 等小新批准

---

## 目標

小新問「今天有什麼建議」時，小馬即時讀取現有資料產出 2-3 條可執行建議。

## 範圍

| 項目 | 做 | 不做 |
|:-----|:---|:-----|
| 資料來源 | finance_activities + finance_budget_alerts + ai-get-debt-summary | 不新增 schema |
| 觸發方式 | 小新對話詢問（「今天建議？」） | 不自動推送 |
| 建議格式 | 2-3 條，優先：超支→債務→異常 | 不做複雜 AI |
| 輸出方式 | 對話回覆 | 不新增 UI |

## 建議規則

1. 已超支或月底預估超支 → 提醒
2. 7 天內債務到期/逾期 → 提醒
3. 分類支出 > 上月 2x 且差額 > $500 → 提醒
4. 今日剩餘可花額 → 建議

## 不動

- ❌ 不新增 DB table
- ❌ 不新增 Edge Function
- ❌ 不改前端
- ❌ 不新增 cron job
