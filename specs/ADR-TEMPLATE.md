# ADR Template — 架構決策紀錄

使用時機：每次做出一個設計決定（選技術、改 schema、定規則）時記錄。

```markdown
# ADR-NNN: [標題]

**日期**: YYYY-MM-DD
**狀態**: [提案中 | 已採用 | 已取代 | 已廢棄]

## 背景

為什麼需要做這個決定？有什麼上下文？

## 選項

- **A**: [方案名稱] — 簡短描述
- **B**: [方案名稱] — 簡短描述
- **C**: [方案名稱] — 簡短描述

## 決策

選擇了 **A**。原因：
1. ...
2. ...
3. ...

## 影響

- 正面：...
- 負面：...
- 需要跟進：...

## 替代方案不選原因

- B：因為 ...
- C：因為 ...

## 參考資料

- [連結或文件]
```
